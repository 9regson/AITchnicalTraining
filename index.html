데이터 파일: 예술의전당 2024 기획 공연 '오페라 관객 통계'

1. 수업 목표

  • 판다스(Pandas)를 활용하여 데이터 전처리의 기본(결측치 처리, 중복 데이터 제거, 데이터 타입 변환)을 실습한다.
  • 파인만 기법을 통해 데이터 전처리를 쉽게 이해하고 설명할 수 있도록 한다.

전개 (30분)

  1. 결측치 처리
    파인만식 설명: "데이터에 빈칸(결측치)이 있으면, 마치 시험지에 이름을 안 쓴 것과 같아. 분석할 때 문제가 생길 수 있으니, 빈칸을 채우거나 빼줘야 해!" 
    # 결측치 확인
print(df.isnull().sum())

# 결측치 채우기 (예: 0으로)
df = df.fillna(0)

# 결측치 행 삭제
df = df.dropna()
    학생 활동: 결측치가 있는지 확인하고, 직접 채우거나 삭제해보기
  2. 중복 데이터 제거
    파인만식 설명: "같은 줄이 두 번 있으면, 시험지를 두 번 낸 것과 같아. 결과가 이상해질 수 있으니, 중복된 줄은 하나만 남기자!" 
    # 중복 데이터 확인
print(df.duplicated().sum())

# 중복 데이터 제거
df = df.drop_duplicates()
    학생 활동: 중복된 데이터가 있는지 확인하고, 직접 제거해보기
  3. 데이터 타입 변환
    파인만식 설명: "숫자가 문자로 저장되어 있으면, 계산이 안 돼! 숫자는 숫자답게, 날짜는 날짜답게 바꿔줘야 해." 
    # '관객수'를 숫자로 변환
df['관객수'] = df['관객수'].astype(int)

# '일자'를 날짜 타입으로 변환
df['일자'] = pd.to_datetime(df['일자'])
    학생 활동: 데이터 타입을 직접 변환해보기
